How it works

  1. Select a product lot and an option: 10% target (3 mo) or 5% guaranteed (6 mo).
  2. Review the profile card with costs, timeline, and safeguards.
  3. Sign the agreement and fund via wire/ACH to a dedicated account.
  4. Track production, shipping, and sales milestones in updates.
  5. Receive principal + return at maturity (or earlier if sell-through completes).

Guarantee terms, if selected, are defined in the signed agreement (e.g., inventory pledge/reserve).

Safeguards

  • Product-level budgets & margin analysis
  • Milestone disbursements tied to PO, QC, logistics
  • Inventory/receivables pledge
  • Periodic reporting; third-party reconciliations possible
